# StockWatch
An Android application to monitor stocks

# Technology Stack
• Java  
• REST API  
• JSON  

***

## Features Implemented

• This app allows the user to display a sorted list of selected stocks. List entries include the stock symbol (and company name), the current price, the daily price change amount and price percent change  
• There is only one layout for both Portrait and landscape orientation in this application – the same layout works in any orientation  
• Selected stock symbols and the related names will be stored in the device’s SQLite Database  
• A Stock class created to represent each individual stock in the application. Data includes: Stock Symbol, Company Name, Price, Price Change, and Change Percentage
• Clicking on a stock opens a browser displaying the Market Watch web page for that stock  
• Swipe-Refresh (pull-down) refreshes stock data  
• In Case of No Internet Connection Swipe-Refresh (pull-down) will display a Pop Up and restarting application will display all stock values zero in Red Colour
